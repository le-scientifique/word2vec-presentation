NLP ⋂ ML

“AI = building systems that can do intelligent things
NLP = building systems that can understand language ⊊ AI
ML = building systems that can learn from experience ⊊ AI
NLP ⋂ ML = building systems that can learn how to understand language”

Typical machine learning pipeline for text analysis.

  1. Gather Input Data - Text in HTML, PDFs, Docx, Audio Transcripts
  2. Text Extraction using HTML Parsers, OCR, Speech2Text etc.
  1. Text Feature Extraction
    • Bag-of-words/n-grams Representation
      • Count Vectors
      • TF-IDF Vectors
    • Word Vectors
  2. ML Algorithms for Clustering, Classification
  3. Rinse, Lather, Repeat
